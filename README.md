# btab

Blue team analyisis box is a tool for blue team security analyisis.

BTAB (Blue Team Analyisis Box) 是一个开源蓝队分析工具箱，辅助安全运营人员的流量分析、木马分析等场景，目前已集成流量包检测、SQL注入检测、Webshell 检测、bash命令执行检测等工具。
